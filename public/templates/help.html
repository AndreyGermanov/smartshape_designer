SmartShape Studio

The SmartShape Studio is an editor to create collections of SmartShapes and save them to JSON files. Then, these shapes can be loaded from these JSON files using "fromJSON" function to the code of JavaScript projects.

Basic functions

  • Use button to add new shapes.
  • Using the editor panel, modify the shape by moving, resizing, rotating it and dragging the points of the shape.
  • To add new point, double-click on the shape.
  • To delete the point, press right mouse button on it.
  • To save created shapes to a file, press button.
  • To load previously saved shapes, press button and select the JSON file.

In the future, this editor will allow not only create shapes from scratch, but also import vector graphics from files of various formats, like SVG or Web fonts. Then, you could modify loaded shapes and save them as a collections of SmartShapes to use in your code.

Please, visit the GitHub repository of SmartShape Studio Project to see the last changes and the most recent documentation.
