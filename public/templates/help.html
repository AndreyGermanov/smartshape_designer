SmartShape Studio

The SmartShape Studio is an editor to create collections of SmartShapes and save them to JSON files. Then, these shapes can be loaded from these JSON files using "fromJSON" function to the code of JavaScript projects.

Basic functions

  • Use button to add new shapes.
  • Using the editor panel, modify the shape by moving, resizing, rotating it and dragging the points of the shape.
  • Use context menu by right mouse click on the shape to add points, delete points and do other modifications
  • Add new figures to the shape, using hamburger menu on top right corner of the editor
  • Using shape context menu, export created shapes to one of available formats (like SVG or PNG).
  • To save created shapes to a file, press button.
  • To load previously saved shapes, press button and select the JSON file.
